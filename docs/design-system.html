Design System · extracted from the app

An iOS-native, glass-first system

This reference documents the design tokens and component patterns currently used in the app — colors, typography, spacing, corner radii, the liquid-glass material, and core controls. Values are pulled directly from contexts/ThemeContext.tsx and the component styles. Flip the toggle to see the dark theme.

Foundations

Spacing

A loosely 2-based scale. gap: 12, gap: 8, and paddingHorizontal: 20 dominate layouts.

Foundations

Corner radius

Continuous (squircle) curves throughout via borderCurve: "continuous". Key roles: buttons 14, cards/glass 16, list rows 12, large cards 24, pills 999.

Material

Liquid glass

The GlassContainer primitive renders four intensities. On supported iOS it uses the native glass effect; elsewhere it falls back to a blur + translucent underlay. Underlay opacity and blur intensity per level shown below.

Clear
underlay .58 · blur 35
Light
underlay .74 · blur 45
Medium

Heavy
underlay .96 · blur 90

Hairline border (StyleSheet.hairlineWidth): rgba(0,0,0,0.08) light / rgba(255,255,255,0.12) dark. Light theme adds a soft fallback shadow 0 1px 3px rgba(0,0,0,0.06).
