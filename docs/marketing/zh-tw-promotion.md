## 品牌核心定位

**一句話：** 追蹤無感消費，零摩擦記帳，重掌財務主導權。

**Tagline 選項：**

1. 別再讓幽靈消費，悄悄掏空你的荷包
2. 無感消費，才是你戶頭裡最厲害的隱形刺客
3. **微小支出**正在悄悄掏空你

## Hook 1：無感漏財

> 這個月沒有大手筆，為什麼戶頭還是這麼少？

### 主標

**沒花大錢，錢怎麼不見了？**

### 副標

小額消費、隨手一刷、日常瑣碎支出——這些 **無感漏財**，才是戶頭變薄的真正元兇。

### 長文案

你記得那筆 3,000 元的晚餐，  
但記不得這週七杯手搖、四次外送、兩次加購。

這就是 **幽靈消費**：  
每一筆都不大，加起來卻很痛。

**黑白記帳** 幫你追蹤 **無感消費**——  
三秒記一筆，不用複雜分類，  
支出立刻變得看得見、算得清。

**零摩擦記帳 → 重掌財務主導權 → 無感存錢**

### 短文案（社群 / 廣告）

- 沒大買，戶頭卻變少？你漏記的不是大筆，是 **無感漏財**。
- 手搖、外送、小額訂閱—— **幽靈消費** 正在悄悄掏空你。
- 記帳不用苦。三秒一筆，把 **無感消費** 全部抓出來。

## Hook 2：反摩擦（Anti-Friction）

> 討厭傳統記帳 App 的繁瑣與干擾。

### 主標

**記帳 App 很煩？這個不會。**

### 副標

沒有複雜報表、沒有惱人提醒、沒有打開就讓你想關掉的介面。  
**零摩擦記帳**——隨手記錄，生活不打斷。

### 長文案

你試過記帳 App，然後放棄了，對吧？

因為它們要你：

- 選分類、填備註、對帳單、看廣告
- 每天提醒「你今天還沒記帳喔！」
- 打開 30 秒，還在找要按哪裡

**黑白記帳** 走完全相反的路：

- **極簡介面** — 金額、描述，完成
- **零摩擦記帳** — 三秒記完，無多餘步驟
- **穩私第一** — 離線能用，資料永遠在你手上

你不需要「更努力的記帳」，  
你要的是 **不用被記帳綁架** 的記帳方式。

### 短文案

- 討厭記帳 App？這個 **不複雜、不煩人**。
- 記帳該像呼吸一樣自然——**零摩擦記帳**，記完就關。
- 別再為記帳開 App 開到心累。三秒，搞定。

### 對比句（適合 Carousel / 圖文）


| 傳統記帳 App    | 黑白記帳        |
| ----------- | ----------- |
| 打開就一堆分類     | 金額 + 描述，完成  |
| 每天催你記帳      | 你想記才記       |
| 介面複雜像 Excel | 極簡，一眼看懂     |
| 資料綁在雲端      | 本機優先，可選雲端同步 |

## Hook 3：訂閱陷阱

> 每月悄悄扣款的訂閱，像寄生蟲一樣吸走你的錢。

### 主標

**訂閱疲勞？該清寄生蟲了。**

### 副標

Netflix、Spotify、iCloud、某健身 App……  
每個月自動扣，你早就忘了為什麼訂。

### 長文案

**訂閱疲勞** 是真實的。

你以為自己沒花什麼，  
但信用卡帳單上躺著 5、8、12 個自動扣款——  
每一個都不多，加起來卻是一筆 **幽靈消費** 大戶。

**黑白記帳** 幫你建立 **消費防火牆**：

- 把固定訂閱當成「支出」記下來
- 每月統計一眼看清 recurring 總額
- 發現「原來我在養這麼多 App」的瞬間

**殺掉寄生蟲，重掌財務主導權。**

*（即將推出：收據掃描 + OCR，自動填入金額與商家）*

### 短文案

- 每個月自動扣，你卻以為沒花錢？這就是 **訂閱疲勞**。
- App 安靜地吸你的血——該把 **寄生蟲訂閱** 清掉了。
- 訂閱一筆一筆記下來，你會嚇到原來每月漏多少。

## 投放注意事項

1. **「不用開 App 就能記帳」**：目前產品核心是「打開後三秒記完」的極簡體驗；若廣告強調「背景自動記帳」，需等 Widget / OCR / 自動化功能上線，或改寫為「快到像沒在記帳」。
2. **訂閱追蹤**：現版可透過「固定支出記錄 + 統計」達成；若要做「自動偵測訂閱」，建議標示「即將推出」或放在 roadmap 文案。
3. **Pro 定價**：商店文案可補上「月付 / 年付解鎖雲端同步」；定價以 App Store Connect 實際設定為準。
4. **香港微調**：「手機 App」→「應用程式」、「匯出」→「導出」即可；其餘文案台港通用。
